## Installation And Run

### Backend (NodeJS)

```bash
$ yarn | npm install
```

## Running the app

```bash
# development
$ yarn start | npm run start

# watch mode
$ yarn start: dev | npm run start:dev

# production mode
$ yarn start:prod | npm run start:prod
```

### Frontend (ReactJS)

```bash
$ yarn | npm install
```

## Running the app

### With Docker

```bash
$ cd compose && docker-compose up --build
```

### Local (PostgreSQL needed)

```bash
# development
$ yarn start | npm run start

# watch mode
$ yarn start:dev | npm run start:dev
```
